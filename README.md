# CMS Image Ratio Guide

An interactive tool for content teams to preview, crop, and validate images across common CMS component aspect ratios — before they go live.

## What It Does

Upload any image and see exactly how it will render inside real CMS layouts. Each component type has a dedicated crop tool with live mockup preview, so you can catch framing issues early.

### Supported Components

| Component | Aspect Ratio | Typical Use |
|-----------|-------------|-------------|
| Hero Banner | 16:9 | Homepage heroes, landing pages, feature announcements |
| Card (Square) | 1:1 | Product cards, team profiles, social previews |
| Card (Landscape) | 16:9 | Blog cards, news feeds, related content |
| Thumbnail | 16:9 | Media libraries, video thumbnails, search results |
| Section | 3:2 | Editorial imagery, case studies, portfolios |
| Cinematic | 21:9 | Full-width banners, immersive storytelling sections |

### Features

- **Interactive crop tool** — drag to reposition, zoom slider to adjust framing
- **Rule-of-thirds grid** — toggle overlay to check composition
- **Live mockup preview** — see the cropped result inside a realistic CMS component
- **Specifications panel** — recommended dimensions, minimum sizes, file size limits, and content guidelines for each ratio
- **Image upload** — test with your own images or use the built-in sample
- **Dark mode** — toggle between light and dark themes
- **Fully responsive** — works on desktop and mobile

## Getting Started

Open `index.html` in a browser, or visit the hosted version (https://renzio.github.io/cms-image-ratio-guide/#/). No build step or server required — it's a static site.

## Built With
